# Star Wars

Crea una página web que muestre los primeros 10 personajes del API de Star Wars (https://swapi.dev/). Únicamente has de mostrar el nombre del personaje y una foto.

Para tal fin deberás agregar un fichero `.js` y, opcionalmente, un fichero CSS con los estilos que prefieras.

-   Bonus: muestra los personajes de todas las páginas.
