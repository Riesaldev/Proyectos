# Rick & Morthy

Crea una página web que muestre los primeros 20 personajes del API de Rick & Morthy (https://rickandmortyapi.com/). Únicamente has de mostrar el nombre del personaje y una foto.

Para tal fin deberás agregar un fichero `.js` y, opcionalmente, un fichero CSS con los estilos que prefieras.

-   Bonus: muestra los personajes de todas las páginas.
