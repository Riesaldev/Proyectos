# Estudiantes

Crea una página web que muestre el nombre, la edad y una foto de cada estudiante. Para las fotos, puedes usar cualquier imagen descargada de la web. Si queréis, podéis usar la misma foto para todos.

Opcionalmente, un fichero CSS con los estilos que prefieras. Podrías intentar crear un <li> con forma de tarjeta para mostrar la info de cada estudiante.
